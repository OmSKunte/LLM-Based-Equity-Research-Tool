LLM-Based Equity Research Tool

An AI-powered document and web intelligence tool that enables users to upload files or analyze web content, ask contextual questions, and generate concise summaries in real time.

The system leverages Natural Language Processing (NLP) and Large Language Model (LLM) APIs to extract meaningful insights from complex financial and textual sources.

Project Overview:
This tool allows users to:
Upload PDF, CSV, or text documents
Analyze web articles via URLs
Ask contextual questions based on uploaded content
Generate real-time AI-powered summaries
Extract insights from complex financial data

Tech Stack:

Python
NLP
LLM APIs
Web Scraping
Streamlit
Pandas / NumPy

Features:

Document upload and processing
Web content extraction
Context-aware Q&A system
Real-time summarization
Clean dark-themed UI
Interactive chat interface
